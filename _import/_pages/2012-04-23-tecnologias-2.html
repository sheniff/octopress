Tecnologías que manejo

Son muchas las tecnologías que han pasado por mis curiosas manos. Y aunque en algunas de las abajo listadas indique que poseo un nivel más avanzado que en otras, ni que decir tiene que de todas debo seguir aprendiendo, ya que es el nuestro un campo de continuo aprendizaje.
  • Desarrollo web de cara al cliente (FrontEnd)
    • JavaScript (Nivel Alto, experimentado en el manejo de JQuery, JQuery UI y frameworks MVC como BackBoneJS)
    • Diseño web con HTML y CSS (Nivel Avanzado, incluyendo frameworks CSS como "Bootstrap from Twitter" para agilizar el trabajo)
    • HTML5 (Nivel Medio, en aprendizaje constante)
  • Desarrollo web de cara al servidor (BackEnd)
    • Python & Django (Nivel Alto)
    • PHP con MySQL (Nivel Medio)
    • Ruby & Rails (Nivel Básico, en aprendizaje)

En el tintero

Me encanta probar nuevas tecnologías y métodos de desarrollo, especialmente si están relacionadas con el desarrollo web. Estas son algunas de las que tengo en lista de espera:
  • Ruby & Rails (quiero seguir profundizando en este potentísimo framework)
  • NodeJS
  • ExtJS
  • HTML5 (De una extensión cada vez mayor, me interesa especialmente las nuevas herramientas JavaScript que ofrece, entre ellas las WebApps)
  • Responsive Design (Me apasiona la posibilidad de adaptar dinámicamente las interfaces al tamaño de la pantalla que las visualiza)
  • SenchaJS, LungoJS (Desarrollo de interfaces para dispositivos móviles, el futuro)
  • Y un largo etc...
